1. Problem Statement: Children‚Äôs Card Game

After your success with creating a toy solar system, you receive an invitation from a toy company, ‚ÄúWizards of the Rich!‚Äù. The company needs your help to create a children‚Äôs card game. But this company has no experience making one before (but that does not stop them because they want cash money). So, you decide to make a simple card game for them.

1.1 Gameplay

In your prototype, a player duels a fixed opponent who never plays any card, which means most of the card functions will only be used by the player.

1.1.1 Game Component

1.Player Character Card

‚Äì At the start of the game, each player will have a default character as an avatar. You choose Player Character before you choose your deck.

Each has different stats of:

‚Äì Life Point = if Life Point of the character in played reaches 0, the corresponding player loses the game.

‚Äì Attack Point = The amount of damage this card deals to the opponent when attacking.

‚Äì Defense Point = The amount of damage this card reduces when defending against an opponent‚Äôs attack.

2.Deck

‚Äì Contains as many cards as you want, you can insert and remove any card at the card shop. However, you can only have up to 4 of the same cards.

3. Card

3.1 Character Card

‚Äì Similar to player character, Character card contains Life Point, Attack Point, and Defense Point.

Player can choose to perform 1 of 2 actions when playing this card.

1. Switch Character

‚Äì All of current character‚Äôs equipped items will be retired. ‚Äì Replace current character in play with this card.

‚Äì Remaining Life Point percentage from previous character will be transfered to new character. (Ex Current character LP = 75/100 -&gt; New character = 112/150) 2. Sacrifice

‚Äì With the cost of this card, you can increase current character Life Point (by percentage). Different types of card increase different percentage of Life Point.

3.1.1 Basic Character Card

‚Äì Character card with only switch and sacrifice actions.

‚Äì Sacrificing it heals the current character for 1/6 of the current character‚Äôs max LP.

3.1.2 Main Character Card

‚Äì Character card that can level up when this card is in play and Player uses the same card. Level up increases all 3 of character stats. ‚Äì Sacrificing it heals the current character for 1/8 of current character‚Äôs max LP.

3.1.3 Exodia Character Card

‚Äì Character card that when equip with 4 Exodia Part Cards (Item Card), Player will immediately win the game.

‚Äì Sacrificing it heals the current character for 1/10 of the current character‚Äôs max LP.

3.2 Item Card

‚Äì Item Card is used to increase character stats, it contains LP bonus, Attack bonus, and Defense bonus.

‚Äì Item Card only has 1 action when played, equip item, which increases the current character‚Äôs stats. Character can have any number of item cards equipped, but all equipped item cards will be lost when player switches character.

3.2.1 Basic Item Card

‚Äì Item card with only previously mentioned function.

3.2.2 Exodia Part Card

‚Äì Item card that only increases character Defense Point, if this card is equipped on to Exodia Character Card, Defense Point increase will be doubled.

1.1.2 Game Flow

At the start of the game, the player shuffles his deck and draw 5 cards. A player character card is played as a placeholder character.

1.Start with the player turn, the player draws a card.

2.Player chooses to play one card and pick its action.

3.Player character automatically attacks opponent character:

‚Äì Damage dealt = Attacker‚Äôs Total Attack Point ‚Äì Defender‚Äôs Total Defense Point

(Damage can‚Äôt be negative)

4.Start the opponent turn, opponent character automatically attacks player character.

5.Repeat until win condition is met by either player.
